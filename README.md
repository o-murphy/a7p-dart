# a7p

**This repository has moved and is archived.**

`a7p` (Dart) is now part of the [a7p](https://github.com/o-murphy/a7p) monorepo, which hosts the `.a7p` ballistic profile format implementation across multiple languages. The Dart package now lives at:

👉 **[github.com/o-murphy/a7p/tree/master/dart](https://github.com/o-murphy/a7p/tree/master/dart)**

This repository is archived (read-only) and no longer receives commits, issues, or pull requests. Please update your bookmarks, forks, and issue trackers to point to the new location. The `a7p` pub.dev package will continue to be published from the new repository.

## Installation

```yaml
dependencies:
  a7p: ^0.1.0
```
